# YM2149-WASM Examples

Die vollständige Dokumentation, API-Beschreibung und Test-Anleitung steht in `crates/ym2149-wasm/README.md`. Dieses Verzeichnis enthält nur die fertigen Demo-Dateien.

Schnellstart:
- `./start-server.sh` ausführen (in diesem Ordner)
- Browser auf `http://localhost:8000/simple-player.html` öffnen

Zum Neu-Build der WASM-Artefakte aus dem Workspace-Root:
```bash
scripts/build-wasm-examples.sh --release
```

Build from Source

Prefer to compile yourself? With Rust installed, you can build the latest version from source or install the CLI directly.

Quick Install: CLI Replayer

If you already have Rust installed, you can install the CLI replayer directly from crates.io with a single command: 

cargo install ym2149-replayer-cli

This installs the ym-replayer binary to your Cargo bin directory (usually ~/.cargo/bin).

Step 1: Install Rust

If you don't have Rust installed yet, the easiest way is via rustup - the official Rust toolchain installer:

# macOS / Linux 
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
# Windows

Download and run rustup-init.exe

After installation, restart your terminal and verify with rustc --version.

Step 2: Clone and Build

Clone the repository and build the binaries you need:

# Clone the repository 
git clone https://github.com/slippyex/ym2149-rs.git
cd ym2149-rs
# Build CLI replayer 
cargo build --release -p ym2149-replayer-cli

Binary: target/release/ym-replayer

# Build Bevy examples 
cargo build --release -p bevy_ym2149_examples --example advanced_example
cargo build --release -p bevy_ym2149_examples --example demoscene

Binaries: target/release/examples/advanced_example, target/release/examples/demoscene

# Or run directly without building first 
cargo run --release -p bevy_ym2149_examples --example advanced_example
